1. By aktywować środowisko wirtualne wejdź za pomocą cmd do folderu ../scripts i wpisz 'activate'
2. Jeśli coś nie zadziała ze względu na brak biblioteki odnajdź plik 'requirements.txt' i w cmd (w odpalonym środowisku wirtualnym) wpisz: pip install -r requirements.txt
3. Tworzenie pliku requirements.txt w cmd po aktywacji wirtualnego środowiska: pipdeptree > requirements.txt
